How to compile TCP connection:

1. Run Sender.java
2. Type "CONNECT_TCP" and press Enter
3. Run Receiver_TCP.java
4. Type "SEND" and press Enter
5. Type desired filename and press Enter


How to compile UDP connection:

1. Run Receiver_UDP.java
2. Run Sender.Java
3. Type "CONNECT_UDP" and press Enter
4. Type "SEND" and press Enter
5. Type desired filename and press Enter
